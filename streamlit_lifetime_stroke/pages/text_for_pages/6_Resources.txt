## About the app  

The code behind this app is written in `python3` and the app itself is deployed using `streamlit`, which is a free service for hosting [python web apps](https://streamlit.io/). 

The code in this app is open-source and available on GitHub:
+ [![GitHub Badge][github-img]][github-link] __Source code:__ https://github.com/stroke-optimist/streamlit_lifetime_stroke

The README file on GitHub contains instructions on where to find the python code for the models and constants.
